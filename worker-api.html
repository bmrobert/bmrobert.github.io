Worker Demographics API Overview

This API retrieves demographic information for employees of Constellation Brands and 3rd party partners. This includes:

  • Constellation Brands employees
  • Contingent workers
  • Distributor and customer employees

The API supports retrieval of all employees’ information, specific employees, and employees added/modified within specific date range. The API doesn’t provide the entire dataset in one call, but does support pagination. Consumers can dictate how many records they need to retrieve.
