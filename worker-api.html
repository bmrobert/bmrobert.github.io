Worker Demographics API Overview

This API retrieves demographic information for employees of Constellation Brands and 3rd party partners. This reference includes:

  • Worker Attributes
  • Job Attributes
  • Work Contact Information
  • Personal Contact Information
  • API Console

The Worker API supports retrieval of all employees’ information, specific employees, and employees added/modified within specific date range. The API doesn’t provide the entire dataset in one call, but does support pagination. Consumers can dictate how many records they need to retrieve.

Request an API Key

Worker Detail

The Worker Demographics API returns a number of attributes for a given employee. This includes personal info, job info, and contact info.

Worker Attributes

Attribute Description Type
ID 7-digit unique employee ID string
workerType "Employee", "Contingent Worker", or "Distributor" string
firstName Legal first name string
middleName Legal middle name or initial (whichever is supplied in the worker's record) string
lastName Legal surname string
preferredName Concatenated full preferred name (First Middle Last) string
dateOfBirth Birth date (YYYY-MM-DD) string
timeZone Time zone of the worker's work location string
preferredLanguage Language specified by the worker as preferred for user interfaces string
createdOnDate Date and time the worker's user account was created (YYYY-MM-DD:hh:mm.ssss) string
activeDirectoryDomain Worker's domain in Constellation Brands Active Directory (if applicable) string

Job Attributes

Attribute Description Type
code 7-digit job code string
title Worker's job title as assigned by HR string
status "Active", "Terminated", or "On Leave" string
isManager True/false flag indicating if the worker has any direct reports boolean
organizationName HR name for the worker's supervisory organization string
managerId 7-digit worker ID of the worker's primary manager string
managerName Concatenated preferred name of the worker's primary manager string
company Legal parent company for the specified worker string
department Worker's department in the reporting hierarchy (one level above Organization) string
division Worker's division in the reporting hierarhy (one level above Department) string
costCenterId Numeric identifier for the worker's cost center string
costCenterName Descriptor for the worker's cost center string
startDate First day of work for the worker in their current position (YYYY-MM-DD) string
